# Selenium Tests
Automated browser test scripts created using *Selenium WebDriver* in *Python*

# About
This repository contains multiple automation scripts built with *Selenium*, designed to perform UI interactions and validations on different web pages. These scripts demonstrate how Selenium can be used to automate common web tasks and functional UI flows.

# Features
-  Automates browser interactions using *Selenium WebDriver*
-  Written in *Python*
-  Test scripts for login flows, form interactions, CRUD operations, etc.
-  Easy to extend with more tests

Getting Started
Prerequisites
Python 3.x
Selenium package
WebDriver for your browser (e.g., ChromeDriver for Chrome)

Install Selenium using pip:

pip install selenium
Usage
Clone this repository:
git clone https://github.com/Sapkotasumita/Selenium.git
Navigate into the project folder:
cd Selenium
Run a test script:
python final.py

Replace final.py with the script you want to run.

Recommended Setup
Use a virtual environment for dependency management:
python -m venv .venv
source .venv/bin/activate   # Linux/macOS
.venv\Scripts\activate      # Windows
Install dependencies in the virtual environment:
pip install -r requirements.txt
